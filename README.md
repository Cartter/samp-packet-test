# samp-packet-test

send packets to a samp server and print the result.

- **USAGE:** 
```python
python samp.py -t 127.0.0.1 -p 7777
```

- **INSTALL PYTHON:** 
```diff
sudo apt-get install python3
```

- **TESTED PACKAGES:** 
```diff
081e77da - (cookie port 7777)
53414d5090d91d4d611e700a465b00 - (query any port)
```
